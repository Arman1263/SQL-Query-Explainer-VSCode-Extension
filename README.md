# SQL Query Explainer

A VS Code extension that converts selected SQL queries into plain-English explanations using AI.

## 🚀 Features

* Right-click to explain SQL queries
* Works directly inside VS Code
* AI-powered explanations
* Fast and lightweight

## 🧪 How to Use

1. Open a `.sql` file
2. Select a SQL query
3. Right-click → **Explain SQL Query**
4. View explanation in the Output panel

## 📦 Installation

Install using the provided `.vsix` file via:

Extensions → ⋯ → Install from VSIX
